This lab focused on user interaction and simple state changes in the app.

This lab improved the app design using Material components and better UI styling.

Lab 5 — Room

This lab introduced Room Database for local data storage.

Reflection: I learned how to save data permanently on the device using Room. This helped me understand local persistence and how apps can work even after reopening.

Project Submissions

Project 1 — WellTrack

Problem Statement: Students may forget to monitor daily health habits such as water intake, sleep, and exercise. WellTrack provides a simple multi-screen app to help record and review these habits.

Lessons learned: In Project 1, I learned how to combine UI, navigation, and ViewModel state management. I also learned how to organize app screens based on one SDG theme.

Project 2 — WellTrack Advanced

Problem Statement: Students need a health tracking app that can store data locally, connect to cloud storage, fetch internet data, and use phone hardware. Project 2 upgrades WellTrack with Room, Firebase Firestore, Web API, and accelerometer sensor integration.

Lessons learned: In Project 2, I learned how to connect different technical features into one complete app. I practiced Room Database, Firebase Firestore, REST API data loading, and SensorManager for accelerometer data.
